# Corso di Git e GitHub per il Liceo Scientifico

Un corso pratico di 9 lezioni pensato per studenti di liceo scientifico che vogliono imparare a usare Git e GitHub partendo da zero.

Ogni lezione dura circa **1 ora** ed è strutturata come una guida step-by-step che gli studenti possono seguire in autonomia, con esercizi pratici e spiegazioni chiare.

## A chi è rivolto

Questo corso è pensato per:

- Studenti di liceo scientifico (classi 3a-5a) senza esperienza precedente con Git
- Docenti che vogliono introdurre il controllo versione nelle ore di informatica
- Chiunque voglia imparare Git partendo dalle basi

## Prerequisiti

- Un computer con accesso a internet
- Nessuna esperienza di programmazione richiesta (anche se aiuta!)

## Come usare questo corso

Ogni lezione è un file Markdown nella cartella `lezioni/`. Gli studenti possono:

1. **Iniziare dalla Lezione 00** per configurare il proprio ambiente di lavoro (Codespaces o VS Code locale).
2. **Leggere le lezioni direttamente su GitHub** cliccando sui link qui sopra.
3. **Seguire gli esercizi passo passo** sul proprio computer.
4. **Procedere al proprio ritmo**, rileggendo le sezioni se necessario.

Le lezioni sono collegate tra loro con link "Lezione precedente" e "Prossima lezione" in fondo a ogni pagina.
